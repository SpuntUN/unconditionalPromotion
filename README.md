# Unconditional Surrender
A short simple text game. You will become a low-level member of a spaceship. After the death of the captain, you will have to find a way home.

## Commands
Move <neighbouring location> - Changes your location {}
Take <Item> - Add said item into your inventory
Relinquish <Item> - Drops said item onto the floor
Look <> / <Item/NPC> - Describes current location if empty, or specific item or NPC
Talk <NPC> - Initiates a dialogue
Use <Item> - Uses an available item and or interacts with the environment.
Inventory <> - Returns a list of items in inventory
Exit <> Exits the game
